# 프로젝트명: CoreAuth (Auth0-like IDaaS)

## 1. 목적
- 중소형 병원 및 기업을 위한 고도화된 멀티테넌트 인증 서비스 구축.
- 보안성, 확장성, 사용자 경험(UX) 측면에서 Auth0 수준의 서비스 제공.

## 2. 핵심 기능
- **Multi-tenancy:** 테넌트별 독립된 사용자 공간 및 설정 제공.
- **Universal Login:** 반응형이며 커스터마이징 가능한 로그인/회원가입 UI.
- **Admin Dashboard:** DevExtreme 기반의 가입자 관리 및 로그 분석 대시보드.
- **Security:** RBAC(역할 기반 권한 제어), MFA(다요소 인증), RLS(행 단위 보안).

## 3. 성공 지표
- Supabase RLS를 통한 완벽한 테넌트 격리.
- DevExtreme 컴포넌트를 활용한 제로 구성(Zero-config) UI 제공.
